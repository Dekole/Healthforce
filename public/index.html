Mockup Flow — 5/2/2026

A guided walkthrough of the HealthForce CNO workflow. Follow the steps in order to experience the full flow from daily dashboard check-in through shift handoff and call-out logging.

Workflow Steps
Step Screen Description Type
1 Dashboard →
The CNO's home base. Shows a real-time snapshot of all units, staffing ratios, and alerts. This is where the shift review begins and where capacity issues surface.
 View
2 Unit Detail →
Drill into a specific unit to see granular staffing data, nurse-to-patient ratios, and current assignments. Full detail view for deep inspection.
 View
3 Unit Quick View →
A condensed unit snapshot designed for rapid review. Surfaces the most critical numbers without navigating away from the main flow.
 View
4 Best Option Modal →
When the system detects a staffing gap, it surfaces the single best corrective action. The CNO reviews the recommendation and approves or adjusts it.
 Modal
5 Recommended Actions →
A broader view of all system-suggested actions across units. The CNO can accept, modify, or dismiss each recommendation before committing changes.
 Action
6 Shift Handoff →
End-of-shift summary passed to the incoming charge. Captures structured staffing concerns, open issues, and optional free-text context for the oncoming team.
 Action
7 Log Call-out →
Records a nurse call-out in real time. Triggers the system to recalculate coverage gaps and surface replacement options immediately.
 Log
